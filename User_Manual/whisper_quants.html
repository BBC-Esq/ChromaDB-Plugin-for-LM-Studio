Whisper Ctranslate2 Quantizations

The "base" model is used by default and will automatically download.

To use different models, simply modify line 13 of voice_recorder_module.py. For example, to use the "medium.en" model with "bfloat16 quantization change it to:

("ctranslate2-4you/whisper-medium.en-ct2-bfloat16", device="auto", compute_type="bfloat16")

Remember, you must restart gui.py afterwards.

Tips:

If you speak English, use models ending in .en since they specialize in English.

Generally, the size of the model is most important followed by the quantizations in the following order:

float32, bfloat16, float16, int8_float32, int8_bfloat16, int8_float16, int8.
